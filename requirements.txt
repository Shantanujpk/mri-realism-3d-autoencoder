SYSTEM REQUIREMENTS

Hardware:
- GPU: NVIDIA RTX 6000 Ada (48 GB VRAM recommended)
- CPU: Multi-core processor (8+ cores recommended)
- RAM: 32 GB minimum (64 GB recommended)
- Storage: 100+ GB (dataset + checkpoints)

--------------------------------------------------

Software Requirements:
- OS: Linux (Ubuntu 20.04+)
- Python: 3.10
- CUDA: 12.x / 13.x
- cuDNN: Compatible with CUDA
- Conda: Miniconda / Anaconda

--------------------------------------------------

GPU & Deep Learning Stack:
- NVIDIA Driver (>= 525)
- CUDA Toolkit (>= 12.0)
- cuDNN
- PyTorch (GPU enabled)

--------------------------------------------------

PYTHON DEPENDENCIES

Install using:
pip install -r requirements.txt

requirements.txt:
torch==2.2.0
torchvision==0.17.0
numpy
matplotlib
nibabel
tqdm
scikit-learn
opencv-python

--------------------------------------------------

ENVIRONMENT SETUP

# Create environment
conda create -n mri_realism python=3.10 -y

# Activate environment
conda activate mri_realism

# Install PyTorch (GPU version)
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu121

# Install other dependencies
pip install numpy matplotlib nibabel tqdm scikit-learn opencv-python

--------------------------------------------------

DATASET REQUIREMENTS

Dataset: BraTS 2023 Glioma MRI Dataset
Format: NIfTI (.nii.gz)

Modalities:
- T1
- T1c
- T2
- FLAIR

SUMMARY

This project uses a GPU-accelerated PyTorch pipeline with medical imaging libraries to train a 3D autoencoder for MRI reconstruction efficiently.
